                H O W   T O   B U I L D   C A M S T U D I O

You'll need to download MS SDK 7.0/7.1 from here: http://www.microsoft.com/en-us/download/details.aspx?id=3138

1. Copy the DShow base library from C:\Program Files\Microsoft SDKs\Windows\<version>\Samples\multimedia\directshow\baseclasses and place it in CamStudio2.7_DShow folder.
1. Open CamStudio.sln
2. Using Configuration Manager change configuration to Release
3. Press F7 and wait
